# 상수와 변수

---

## 선언 방법

- 상수는 let으로 선언한다.

```swift
let sum1 = 100
//let 상수명 = 값
sum1 = 102
//상수 위와 같이 값을 변경 할 수 없다.
//값을 변경하기 위해서는 아래와 같이 변수를 사용해야한다.
```

- 변수는 var로 선언한다.

```swift
var sum2 = 101
//var 변수명 = 값
sum2 = 103
//변수는 상수와 달리 값을 변경할 수 있다.
```

- 상수나 변수 선언 후에 값을 할당하기 위해서는 아래와 같이 해야한다.

- 상수
    
    ```swift
    let sum3: Int
    //let 상수명: 타입
    sum3 = 104
    //상수명 = 값
    ```
    

- 변수
    
    ```swift
    var sum4: Int
    //var 변수명: 타입
    sum4 = 105
    //변수명 = 값
    ```
